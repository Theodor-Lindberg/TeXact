\section{Introduction}

This paper discusses implementations on ASIC and FPGA hardware.
We also explore VLSI design techniques and the FloPoCo framework.

\subsection{Incorrect Spellings}

The following will have incorrect spellings:
- asic designs are common
- fpga-design are nice
- We use fpga for rapid prototyping
- VLSI: very large scale integration
- flopoco is a great tool
- AI and ML are important fields
- But sometimes written as: Asic, Fpga, VlSi, flopoCo, ai, ml


\subsection{Correct Spellings}

These are all correct:
- ASIC implementations
- FPGA prototypes
- VLSI technology
- FloPoCo designs
- AI systems
- ML models
